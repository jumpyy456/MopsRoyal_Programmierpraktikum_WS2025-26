# MopsRoyal_Programmierpraktikum_WS2025-26
„Mops Royal“ ist ein strategisches Legespiel für 2–4 Spieler. Jeder hat ein 5×5-Spielbrett und nutzt einen gemeinsamen Plättchenvorrat. Ziel ist es, durch geschicktes Platzieren und das Bilden gleichfarbiger oder gleichartiger Kombinationen Punkte zu erzielen, größere Kombinationen bringen mehr Punkte. Wer am Ende die meisten Punkte hat, gewinnt.
